## 本课知识点回顾

* 什么是**列表（list）**？
    * **一些**按一定**顺序排列**的数字或字符串
* 列表的例子
    * 彩票 （数字列表）
    * 作业列表 （字符串列表）
    * 购物清单 （字符串列表）
* 列表中的每一项叫做**元素**
* 列表中的元素的位置叫做**索引（index）**


### 小技巧：遍历字符串
```python
letters = 'abcdefghijklmnopqrstuvwxyz'

for letter in letters:         
    print('letter')

```

### 小技巧：提取字符串中的字母
```python
letters = 'abcdefghijklmnopqrstuvwxyz'
print(letters[0])
print(letters[1])
print(letters[2])

```
